# Library Inventory Manager

A Python-based command-line Library Management System using Object-Oriented Programming, JSON persistence, and clean modular code.

## Features
- Add books (title, author, ISBN)
- Issue & return books
- Search by title or ISBN
- JSON storage for persistence
- Logging of operations
- Exception handling
- Package-based folder structure

## How to Run
```bash
cd cli
python main.py
